*This repository contains a short paper which was extracted from my Master's thesis, and accepted by LACL 2016.*

## Strong and weak quantifiers in focused NL<sub>CL</sub>

In this paper, I propose an improvement of Barker and Shan's NL<sub>CL</sub> for which derivability is decidable, which has a normal-form for proof search, can analyse scope islands, and distinguish between strong and weak quantifiers.
